# RaphEdge

RaphEdge is a lightweight, high-performance web browser optimized for older hardware. Designed to deliver a smooth browsing experience without the heavy resource footprint of modern browsers, RaphEdge brings speed and efficiency back to your aging PC.

🚀 Key Features
Low Resource Consumption: Optimized to run on limited RAM and older CPUs.

Minimalist Design: A clean interface that doesn't get in your way.

Fast Loading: Engineered to render web content quickly even on slower machines.

Modern Standards: Built on the WebView2 engine to ensure compatibility with modern websites while maintaining efficiency.

🛠 Prerequisites
To run RaphEdge, ensure you have the following installed on your system:

Windows 10/11

📥 Installation
Download the latest release from the Releases page.

Extract the ZIP file to your preferred directory (e.g., C:\RaphEdge).

Run RaphEdge.exe.

💻 Tech Stack
Language: C#

Framework: .NET 6.0

Engine: Microsoft.Web.WebView2
